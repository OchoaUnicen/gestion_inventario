Sistema de Gestión de Stock

Interfaces del Sistema

Inventario

La interfaz de inventario permite visualizar el listado completo de productos registrados. Para cada producto se muestra información clave como:

  • Nombre del producto
  • Código de barras
  • Stock disponible
  • Precio

Escaneo de Productos

Desde esta interfaz se pueden identificar productos mediante el escaneo del código de barras. El sistema permite:

  • Identificar productos existentes
  • Agregar nuevos productos
  • Editar información de productos ya registrados

Movimientos de Stock

La interfaz de movimientos muestra el historial completo de operaciones realizadas sobre el inventario, permitiendo visualizar:

  • Entradas de stock
  • Salidas de stock
  • Producto involucrado
  • Cantidad y fecha del movimiento

Notificaciones

Esta interfaz centraliza las alertas automáticas del sistema, mostrando:

  • Productos con stock bajo
  • Productos sin stock

Esto permite anticiparse a reposiciones y evitar faltantes.

Dashboard

El dashboard ofrece una vista general del estado del sistema mediante indicadores simples (KPIs), pensados para brindar información rápida sin aumentar el costo ni la complejidad del desarrollo.

Inicialmente, el dashboard puede incluir métricas básicas como estado general del stock y alertas activas, quedando preparado para incorporar indicadores más avanzados en futuras etapas.
